# Aurly 🚀

**Aurly** é um sistema de **filas digitais simples, rápido e gratuito**, criado para resolver um problema real:  
a burocracia absurda dos softwares de fila existentes.

Sem cadastro.  
Sem app.  
Sem complicação.

👉 **https://aurly.xyz**

---

## ✨ O que é o Aurly?

Aurly permite que qualquer negócio crie e gerencie uma fila digital em segundos.

O cliente entra na fila pelo celular e acompanha sua posição em tempo real.  
O atendente chama o próximo com um clique.

Tudo online, tudo simples.

---

## 🎯 Problema que resolve

- Confusão na ordem de atendimento  
- Clientes reclamando de espera  
- Sistemas caros e cheios de etapas  
- Dependência de papel, gritos ou senhas físicas  

Aurly elimina tudo isso.

---

## ⚡ Principais funcionalidades

- 📱 Entrada na fila via link ou QR Code  
- 🔢 Acompanhamento da posição em tempo real  
- ⏱️ Estimativa de tempo de espera  
- 🔔 Alerta sonoro para o cliente  
- ▶️ Botão “Chamar Próximo” para o atendente  
- ❌ Cancelamento de senha com um clique  
- 🧠 Atualização em tempo real (sem reload)
